요소를 원하는 위치에 자요롭게 이동시키는 태그 (Type->기준점)
    static = 모든 요소의 기본 포지션 값
    relative = 기준점 : 자기 자신이 본래있었던 위치(static)을 기점으로 이동함. 무난하게 다른 포지션 요소의 기준점으로 사용 / float랑은 무관하게 기존에 부모요소에 종속됨
    absolute = 공중부양 느낌 / (기준점을 다른 postion요소로 잡게됨 static은 제외) float랑 비슷함( 디스플레이가 블럭이 됨 / 그런데 inline요소도 블럭할수 없음)
    fixed = absolute랑 대부분이 동일하지만 , 자신의 부모요소 중에서 다른점(position)을 기준점을 viewport (접속환경의 디스플레이크기)로 잡음
    sticky(아직 지원하는 브라우저가 많지 않으므로 스킵)
    z-index = static을 제외한 다른 포지션요소를 사용한다면 붕뜨게됨 그 공중부양하는 z축을 설정
    ex) A는 상공10m, B는 상공20m 위에 있는 친구가 아래에 있는 친구를 덮으므로 위에 보여줘야할 요소를 높게 설정하는것
